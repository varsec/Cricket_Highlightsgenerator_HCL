# Cricket_Highlightsgenerator_HCL
Project Overview

This project is a Cricket Highlights Generator that extracts important moments such as fours, sixes, and wickets from a full match video and generates a short highlights video.

The system uses event timestamps to identify key moments and automatically creates a summarized video with labels.

Objectives
Reduce long cricket match videos into short highlights
Extract important events like:
Four
Six
Wicket
Add labels to each highlight clip
Generate a clean and professional output video

Technologies Used
Python
MoviePy (Video Processing)
Pandas (Data Handling)
